# Progetto-Python
Progetto Python 3 CFU ICT
Il progetto volgerà ad analizzare un database ritraente un questionario posto a studenti con diversi tipi di domande volte a comprendere se ci sono relazioni tra variabili per definire delle tendenze nelle performance accademiche. In seguito all'analisi del DB, verrà creato un piccolo modello predittivo che prova a predirre la performance futura di uno studente sulla base del proprio allenamento (del modello).
